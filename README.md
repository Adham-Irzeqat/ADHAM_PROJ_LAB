# Smart Task Organizer - منظم المهام الذكي

تطبيق ويب بسيط وسهل الاستخدام لإدارة المهام اليومية. يساعدك على تنظيم مهامك، تتبع مواعيدها، وترتيبها حسب الأولوية.

## المميزات الرئيسية

### المتطلبات الوظيفية (Functional Requirements)

✅ **FR1**: إنشاء مهمة جديدة
- إضافة مهام جديدة مع العنوان، الوصف، الموعد النهائي، والأولوية

✅ **FR2**: تعديل مهمة
- تعديل بيانات أي مهمة موجودة

✅ **FR3**: حذف مهمة
- حذف المهام غير المرغوب فيها

✅ **FR4**: عرض قائمة المهام
- عرض جميع المهام بشكل منظم وواضح

✅ **FR5**: الحالة الافتراضية
- كل مهمة جديدة تكون بحالة "ToDo" افتراضياً

✅ **FR6**: تعليم المهمة كمكتملة
- تغيير حالة المهمة من "ToDo" إلى "Completed" والعكس

✅ **FR6**: الفرز (Sorting)
- فرز المهام حسب الموعد النهائي
- فرز المهام حسب الأولوية

✅ **FR7**: الفلترة (Filtering)
- عرض المهام المكتملة فقط
- عرض المهام غير المكتملة فقط
- عرض المهام ذات الأولوية العالية فقط

✅ **FR8**: الحفظ التلقائي
- حفظ جميع المهام تلقائياً عند إغلاق التطبيق

✅ **FR9**: التحميل التلقائي
- تحميل المهام المحفوظة تلقائياً عند فتح التطبيق

✅ **FR10**: تصدير المهام
- تصدير جميع المهام إلى ملف نصي قابل للقراءة

### المتطلبات غير الوظيفية (Non-Functional Requirements)

✅ **NFR1**: الاستجابة السريعة
- جميع العمليات تستجيب خلال أقل من ثانيتين

✅ **NFR2**: واجهة بسيطة
- تصميم حديث وسهل الاستخدام

✅ **NFR3**: بيانات قابلة للقراءة
- البيانات محفوظة بصيغة JSON في LocalStorage
- إمكانية التصدير لملف نصي منظم

## البنية المعمارية

### Design Patterns المستخدمة

1. **MVC Pattern (Model-View-Controller)**
   - **Model**: `Task.js` - يمثل كيان المهمة
   - **View**: `TaskView.js` - يعالج عرض البيانات والواجهة
   - **Controller**: `TaskController.js` - يدير منطق العمل والتنسيق

2. **Repository Pattern**
   - **TaskRepository.js** - يعزل منطق الوصول للبيانات عن منطق العمل
   - يسهل تغيير طريقة التخزين مستقبلاً (مثلاً: قاعدة بيانات)

## كيفية الاستخدام

### التشغيل

1. افتح ملف `index.html` في متصفح ويب حديث
2. التطبيق يعمل مباشرة بدون الحاجة لتثبيت أي شيء

### إضافة مهمة جديدة

1. املأ حقل "العنوان" (إجباري)
2. أدخل الوصف (اختياري)
3. اختر الموعد النهائي (اختياري)
4. اختر الأولوية (افتراضياً: متوسطة)
5. انقر على "إضافة"

### تعديل مهمة

1. انقر على زر "تعديل" للمهمة المراد تعديلها
2. عدّل البيانات في النافذة المنبثقة
3. انقر على "حفظ التعديلات"

### حذف مهمة

1. انقر على زر "حذف" للمهمة المراد حذفها
2. أكّد الحذف في الرسالة المنبثقة

### تعليم مهمة كمكتملة

- انقر على زر "إكمال" لتغيير الحالة إلى "مكتملة"
- انقر على "إعادة للقائمة" لإعادة المهمة لحالة "ToDo"

### الفرز والفلترة

- استخدم قائمة "فلترة" لعرض نوع معين من المهام
- استخدم قائمة "ترتيب" لترتيب المهام حسب الموعد أو الأولوية

### تصدير المهام

- انقر على زر "تصدير المهام" لتحميل ملف نصي يحتوي على جميع المهام

## التخزين

- البيانات محفوظة في **LocalStorage** للمتصفح
- التخزين تلقائي عند كل عملية (إضافة/تعديل/حذف)
- التحميل تلقائي عند فتح التطبيق
