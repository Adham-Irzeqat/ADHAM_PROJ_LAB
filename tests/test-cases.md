# Test Cases - Smart Task Organizer

## Test Case Documentation

### TC-FR1-001: إنشاء مهمة جديدة
- **Test Case ID**: TC-FR1-001
- **Requirement ID**: FR1
- **Test Objective**: التحقق من إمكانية إنشاء مهمة جديدة بجميع الحقول المطلوبة
- **Preconditions**: 
  - فتح التطبيق
  - التأكد من عدم وجود مهام مسبقاً (أو معرفة العدد الحالي)
- **Test Steps**:
  1. إدخال عنوان المهمة: "مهمة اختبار"
  2. إدخال الوصف: "هذه مهمة للاختبار"
  3. اختيار الموعد النهائي: تاريخ مستقبلي
  4. اختيار الأولوية: "عالية"
  5. النقر على زر "إضافة"
- **Expected Result**: 
  - تظهر المهمة الجديدة في قائمة المهام
  - تحتوي المهمة على جميع البيانات المدخلة
  - حالة المهمة الافتراضية هي "ToDo"
  - يتم تحديث عدد المهام
